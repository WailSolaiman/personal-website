Über mich...

Hallo, mein Name ist Wail Solaiman, ich bin 34 Jahre alt und ich komme aus dem Sudan.

Ich habe an der Fachhochschule Lübeck die Fachrichtung Informatik/Softwaretechnik (B.Eng.) studiert und im Jahr 2013 abgeschlossen. Im Studium habe ich verschiedene Programmiersprachen wie JAVA und C gelernt.

Neben diesen Programmiersprachen habe ich auch Kenntnisse in der Web-Technologien erworben (HTML5, CSS3, und Javascript). Die erlernten Kenntnisse kann ich unmittelbar in kleineren bzw. mittleren Projekten mit verschiedenen Entwicklungsumgebung wie Eclipse, Visual Studio Code oder mit beste Texteditoren wie Atom und Sublimetext in Anwendung bringen.

Derzeit studiere ich den Master Studiengang Medieninformatik (Online) an der Fachhochschule Lübeck.
